css选择器介绍

属性选择器

    除了ie6，都可以支持了
    [attr=value] //该属性表示有确切的值
    [attr~=value] //该属性表示一系列用空格隔开的多个值
    [attr|=value] //该属性以value开头或者以value开头并且立即跟上一个中划线
    [attr^=value] //该属性的值以指定的值开始
    [attr$=value] //该属性的值包含指定的值（无论其位置）
    [attr*=value] // 该属性的值以指定的值结束 
  
      a[href$=".jpg"] {  
        background: url(jpeg.gif) no-repeat left 50%;
        padding: 2px 0 2px 20px;
        }
      
      a[href$=".pdf"] {  
        background: url(pdf.gif) no-repeat left 50%;
        padding: 2px 0 2px 20px;
        }
      
      a[href$=".doc"] {  
        background: url(word.gif) no-repeat left 50%;
        padding: 2px 0 2px 20px;
        }

子选择器

    子选择器用符号“>”表示。它允许你定位某个元素的第一级子元素。

兄弟组合

      该选择器使用加号“+”来链接前后两个选择器。选择器中的元素有同一个父亲，而且第二个必须紧紧的跟着第一个。
      该选择器使用加号“~”来链接前后两个选择器。选择器中的元素有同一个父亲，不同的是第二个选择其无需紧紧跟随第一个。

伪类

      有两类动态伪类：链接 和用户行为。
      链接就是:link 和:visited，
      而用户行为包括:hover、:active 和:focus。

:first-child

ie8及以上

      :first-child伪类允许你定位某个元素第一个子元素。

css3选择器

  • :target
  • :nth-child
  • :nth-last-child
  • :nth-of-type
  • :nth-last-of-type
  • :last-child
  • :first-of-type 和:last-of-type
  • :only-child
  • :only-of-type
  • :empty

否定选择器

      否定选择器:not()，可以让你定位不匹配该选择器的元素

伪元素

    ::first-line伪元素将匹配block、inline-block、table-caption、table-cell等等级别元素的第一行
    ::first-letter伪元素将会匹配一个文本块的第一个字母，除非在同一行里面包含一些其它元素，比如图片。
    ::before和::after 伪元素用于在一个元素的前面或后面插入内容，纯CSS方法。
